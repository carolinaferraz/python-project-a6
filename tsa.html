WHAT IS SENTIMENT ANALYSIS?

In August of 2014, Oxford Dictionaries announced a new batch of additions, amongst them, Sentiment Analysis. The entry states that it is "[t]he process of computationally identifying and categorizing opinions expressed in a piece of text, especially in order to determine whether the writer's attitude towards a particular topic, product, etc. is positive, negative, or neutral." (source)

While the concept might not be exactly new, the tools to perfect the extraction and analysis processes receive a considerable amount of attention from a diverse pool of fields of research. The definition itself hints at sentiment analysis covering reactions to a product, and one could safely infer how that information can be used by the manufacturers in order to improve the product itself or its marketing strategy. Besides that, sentiment analysis is known to aid in areas such as economy, advertising, sociology, and political science.

This paper will explore the use of sentiment analysis in social media as implemented using the Python programming language along with a couple of its downloadable libraries. It will present a simple script as an introduction to the process and briefly point out its inner workings noting Python's flexible syntax as an argument for its wide use for these scripts.

SETTING UP PYTHON-TWITTER INTEGRATION:

The chosen platform to interact with the sentiment analysis script was Twitter due to the shortness of their messages and the tools readily available for this task: Tweepy and TextBlob.

Tweepy "enables Python to communicate with Twitter platform and use its API" and Textblob is a "library for processing textual data". Once Tweepy provides the credentials for accessing Twitter's users' messages (called 'tweets'), Textblob turns them into classifiable elements and starts the analysis stage.

"The sentiment property returns a named tuple of the form Sentiment(polarity, subjectivity). The polarity score is a float within the range [-1.0, 1.0]. The subjectivity is a float within the range [0.0, 1.0] where 0.0 is very objective and 1.0 is very subjective." (source)

Based on the ratings, the script places the tweet into the positive, negative, or neutral categories and prints a small amount of messages for each segment, according to the parameters inserted onto the script's query field of the main function.

It should be noted that this is far from being the only implementation possible, new Natural Languages Toolkit-based tools are being developed for this purpose at a steady pace (VADER Sentiment Analysis is an example).

CONCLUSION:

While the enormous challenge of analyzing natural languages seems daunting, the implementation of analyzers in Python proves to be easy and lightweight.

The shortcomings of the output are expected (given the extremely basic framework and small input) but Python's ability to produce these scripts with ease as well as integrate its built-in string/substring methods for better data management and control justify its extensive use for this purpose.
